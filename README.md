・ゲームの説明/the description of games
1.math_game
このゲームでは、ターミナルに1~10000までの数字の足し算が出てきます。
そこにその足し算の答えを入力してください。
正解すると、次の問題へ行くか、y(yes)/n(no)で聞かれるので、
続けたい場合はyを、もうやめる場合はnを入力して下さい。
このゲームを作って、ヘッダーの基礎的な使い方や、簡単なゲームの作り方を学べました。

There is no description for non-Japanese spearkers yet. sorry. :(

2.clicker_game
このゲームは、opensiv3dで作ったゲームです。
単純なクリッカーゲームで、左クリックするとポイントが増えます。
ポイントは、右上にあるアップグレードボタンを押すと出てくる様々なアップグレードに使うことができます。
アップグレードを閉じるには、右上の「Quit」ボタンを押してください。

There is no description for non-Japanese spearkers yet. sorry. :(

・increaseボタン : クリックで増やすポイントの増加量が増える
・auto clickボタン : ボタンをクリックしなくてもポイントが自動で増えるようになる
・Rebirthボタン : ポイントやアップグレードがリセットされる代わりに、クリック・オートクリックで増えるポイントの倍数が増える

尚、このゲームにはセーブ機能はついていませんので、
セーブ機能をつけるアップデートなどが来ない限り、
一度やめると最初からとなるので、気をつけてください。
